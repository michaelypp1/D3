# D3
Neon Delivery

Neon Delivery is a physics-based Phaser prototype created for D3.

Play:
WASD / Arrow Keys = Move
SHIFT = Boost
E = Grab / Launch
R = Restart

Gameplay Requirements

This project uses both continuous and discrete inputs.

Continuous inputs:
 WASD movement
 Holding SHIFT for movement boost

Discrete inputs:
 Pressing E to grab or launch the energy core
 Clicking to transition between scenes
 Restarting with R

The project uses Phaser Arcade Physics with:
 collisions
 forces
 velocity
 moving obstacles
 bounce interactions

The game contains 3 distinct gameplay scenes:
1. Maze navigation
2. Moving obstacle timing
3. Launch and bounce routing

Summary scenes separate gameplay sections and show completion time.

Assets

All visuals were created using Phaser primitive shapes and text.
No external assets were used.
